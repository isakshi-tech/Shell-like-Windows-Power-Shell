A simple shell built in Node.js that works like a real terminal. You can type commands, and it finds and runs them — just like bash or cmd.

What it does
Takes user input in real time
Runs external programs like node, git, python — anything in your PATH
Comes with built-in commands (echo, exit, type)
Searches your PATH to find executables automatically
Works on Windows (handles .exe, .cmd, .bat)

how to run this on you local system?
1.clone the repository
2.type the command npm run dev
